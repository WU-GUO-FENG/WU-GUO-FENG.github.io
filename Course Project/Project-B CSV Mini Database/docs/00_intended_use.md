# 00_intended_use.md - 預期用途說明書 (Intended Use)

## 1. 專案背景與目的 (Background & Purpose)
在許多輕量級應用、嵌入式系統或邊緣運算環境中，引入大型關聯式資料庫（如 PostgreSQL、MySQL）或 NoSQL 資料庫往往顯得過於臃腫且耗費資源。然而，應用程式依然需要對結構化資料進行有效率的篩選與查詢。

本專案旨在開發一個基於 C++ 的 **CSV 微型資料庫與查詢引擎 (CSV Mini Database & Query Engine)**。它能直接將通用的 CSV 檔案視為資料表，並在記憶體中建立高效能的索引架構，提供簡單的語法進行資料查詢。本專案專注於底層資料結構的應用（如雜湊表、樹狀結構）、記憶體管理以及效能權衡（Trade-offs）的實作。

## 2. 核心功能與範圍 (Scope & Core Features)

### 2.1 內含功能 (In-Scope)
* **健壯的 CSV 解析器 (Robust CSV Parser)**：
  * 支援標準 CSV 格式。
  * 能夠正確處理欄位內包含引號、逗號（例如 `"Huang, Yu-Feng"`）或特殊字元的邊界狀況。
* **記憶體內資料表 (In-Memory Table/DataFrame)**：
  * 將解析後的 CSV 資料映射至高效的記憶體結構中。
  * 支援基本的資料型態識別（如整數、浮點數、字串）。
* **微型查詢引擎 (Simple Query Engine)**：
  * 支援簡單的文字型查詢語法。
  * 支援基本條件過濾（例如 `==`, `!=`, `>`, `<`）。
  * 支援邏輯運算子組合（例如 `AND`, `OR`）。
* **索引機制 (Indexing System)**：
  * 實作雜湊索引（Hash Index）以達到 $O(1)$ 的精準比對速度。
  * 實作範圍索引（Range Index，如紅黑樹）以優化範圍查詢。
* **效能分析與評估**：
  * 提供數據對比「全表掃描（Table Scan）」與「索引查詢（Indexed Search）」在時間與空間複雜度上的差異。

### 2.2 外圍功能 / 暫不支援 (Out-of-Scope)
* 支援標準 SQL 全語法（如複雜的 `JOIN`, `GROUP BY`, `SUBQUERY` 等）。
* 資料寫入與事務處理（ACID Transactions，本專案以 **唯讀查詢 Read-Only** 為主）。
* 圖形化使用者介面 (GUI)，本專案全權基於命令列介面 (CLI) 與 API 呼叫。
* 網路伺服器化（如透過 TCP/IP 進行遠端資料庫連線）。

## 3. 目標使用者與使用情境 (Target Users & Use Cases)
* **資料分析工程師**：需要一個超輕量、無需配置環境的工具，在本地端終端機快速對數十萬行的 CSV 進行特定條件過濾。
* **邊緣運算/嵌入式系統**：在記憶體與效能受限的硬體上，需要對靜態設定檔或紀錄檔（Logs）進行快速索引與查詢。
* **軟體工程學習者**：作為理解資料庫底層設計（Parser、AST、Indexing）與 C++ 效能取捨的經典範例。

## 4. 成功準則 (Success Criteria)
一個成功的專案交付必須滿足以下條件：
1. **正確性**：CSV 解析器必須百分之百正確處理複雜引號欄位，且查詢引擎返回的結果必須與標準資料分析工具（如 Python Pandas）一致。
2. **高效能**：在建立索引後，特定欄位的精準查詢（Equality Query）效能應明顯優於未建立索引前的線性掃描。
3. **穩定性**：專案必須擁有完整的單元測試（Unit Tests），並在 GitHub Actions CI 流程中通過所有編譯與測試，無記憶體洩漏（Memory Leaks）。
4. **文件完整性**：從 `00_intended_use` 到 `08_known_issues` 的所有 SDLC 文件皆撰寫完備，具備良好的可追溯性（Traceability）。

## 5. 環境與設計約束 (Constraints)
* **程式語言**：必須使用 C++17（或更高）標準撰寫。
* **外部依賴**：核心邏輯儘可能使用 C++ 標準函式庫 (STL)，若使用外部輕量庫必須透過套件管理器（如 `vcpkg` 或 `Conan`）進行明確管理。
* **記憶體限制**：資料庫採用全記憶體（In-Memory）運作，處理的 CSV 檔案大小受限於系統可用記憶體實體大小。
